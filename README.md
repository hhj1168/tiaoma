挑码助手 (Macau Picker Web Tool)

### 简介
一个高级的单页 Web 应用 (`挑码助手.html`)，用于六合彩等数字游戏的智能筛选和挑码。设计风格极简且极具现代感。

### 功能特点
*   **多维度筛选**: 支持超过 10 种筛选维度，包括：
    *   **基本形态**: 单双、大小。
    *   **波色**: 红/蓝/绿波及其单双细分。
    *   **生肖/五行**: 12 生肖、金木水火土。
    *   **尾数/头数**: 0-9 尾，0-4 头。
    *   **复合条件**: 大单/小双、合数单双/大小、家禽/野兽等。
*   **智能交互**:
    *   左侧控制面板采用栅格布局系统，美观对称。
    *   点击筛选条件叠加求交集（AND 逻辑）。
    *   实时显示符合条件的号码数量，并自动高亮对应的号码球。
*   **极致体验**:
    *   **年份自动计算**: 内置算法根据年份自动计算当年生肖排位（基准设为 2025 蛇年）。
    *   **一键复制**: 方便地将筛选结果复制到剪贴板。
    *   **响应式设计**: 适配各种屏幕尺寸，支持毛玻璃（Glassmorphism）特效。

### 使用方法
直接在浏览器中打开 `挑码助手.html` 即可使用，无需安装任何服务器环境。
